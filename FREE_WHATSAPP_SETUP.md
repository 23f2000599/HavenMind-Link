# Free WhatsApp Notifications Setup

## Overview
Using **pywhatkit** - completely free, no API keys or registration required.

## Setup Steps

### 1. Install Dependencies
```bash
pip install pywhatkit
```

### 2. How It Works
- Uses WhatsApp Web in your browser
- Automatically opens WhatsApp Web and sends messages
- Requires you to be logged into WhatsApp Web once
- Completely free - no API costs

### 3. First Time Setup
1. Open WhatsApp Web in your browser
2. Scan QR code with your phone
3. Keep browser tab open
4. Test the notification system

### 4. Usage
- Emergency alerts automatically sent via WhatsApp
- Opens WhatsApp Web and sends message
- Works with any phone number
- No monthly fees or API limits

## Limitations
- Requires browser to be open
- Need to scan QR code occasionally
- 1-2 minute delay for message sending
- Computer must be running

## Alternative: Email Only
If WhatsApp setup is too complex, the system falls back to email notifications automatically.
